Lesson 3

Javascript output

Javascript bisa menampilkan data dengan berbagai macam cara yaitu:

  • Menampilkan data ke dalam inner dari tag HTML, menggunakan innerHTML.
  • Menulis data kedalam dokumen dengan menggunakan document.write()
  • Menampilkan data kedalam kotak peringatan browser menggunakan window.alert()
  • Menampilkan data ke dalam console browser, menggunakan console.log()
